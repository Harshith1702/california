========================================
 CALIFORNIA – CRICKET BAT WEBSITE
========================================

OVERVIEW
--------
This repository contains a simple static website built using HTML and images.
It showcases cricket bats through a clean landing page and a secondary
information page.


WEBSITE PAGES
-------------
• index.html  → Landing page
• it3.html    → About / Read More page

Navigation between pages is handled using standard HTML links.


LANDING PAGE (index.html)
-------------------------
Includes:
• Centered logo image
• Main heading introducing cricket bats
• Short descriptive paragraph
• “Read more” button linking to it3.html
• Product showcase section with images
• Closing thank-you message

All styling is done using inline CSS.


ABOUT PAGE (it3.html)
---------------------
Includes:
• Detailed description of cricket bats
• Information about quality and brands
• Motivational cricket-related quote

This page focuses on text content with inline CSS styling.


TECHNOLOGIES USED
-----------------
• HTML
• Inline CSS
• Images (JPG / AVIF)

No JavaScript or backend is used.
